b)
proc sort data = data1; by id; run;
proc sort data = data2; by id; run;

proc sort data=inc_dep1 out=inc_dep2;
update data1 data2;
by id;
run;
proc print data=inc_dep2; run;

c)
merge로 결합한 것은 뒤에 데이터 값을 마지막에 가져오고 update로 결합한 것은
원래의 값을 유지하게 된다

d)
